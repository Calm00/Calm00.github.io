# 费马小定理闪亮登场

定理内容：如果ppp 是一个质数，而整数aaa 不是ppp 的倍数，则有ap−1≡1(modp)a^{p-1}\equiv1\,(mod\,p)ap−1≡1(modp)

# 第一个证明

首先我们考虑一个前置定理：

若gcd(c,p)=1gcd(c,p)=1gcd(c,p)=1，且ac≡bc(modp)ac\equiv bc\,(mod\,p)ac≡bc(modp)，那么有a≡b(modp)a\equiv b\,(mod\,p)a≡b(modp)

证：

∵ac≡bc(modp)∴(a−b)c≡0(modp)∴(a−b)c是p的整数倍又∵gcd(c,p)=1∴a−b≡0(modp)即a≡b(modp)\because ac\equiv bc\,(mod\,p)\\ \therefore (a-b)c\equiv 0\,(mod\,p)\\ \therefore (a-b)c\,是\,p\,的整数倍\\ 又\because gcd(c,p)=1\\ \therefore a-b\equiv0\,(mod\,p)\\ 即a\equiv b\,(mod\,p)∵ac≡bc(modp)∴(a−b)c≡0(modp)∴(a−b)c是p的整数倍又∵gcd(c,p)=1∴a−b≡0(modp)即a≡b(modp)

证毕

# 第二个证明

然后我们进入正题，假设有正整数a(a<p)a(a<p)a(a<p) 满足条件gcd(a,p)=1gcd(a,p)=1gcd(a,p)=1，那么我们将aaa 乘上1∼p−11\thicksim p-11∼p−1 后可以构成一个modpmod\,pmodp 的完全剩余系

证：

假设存在xa≡ya(modp),且x≠y∵gcd(a,p)=1∴原式成立当且仅当x≡y(modp)又∵x,y∈[1,p−1]∴x≡y(modp)当且仅当x=y,与已知条件矛盾∴假设不成立，原命题成立假设存在xa\equiv ya\,(mod\,p),且x\neq y\\ \because gcd(a,p)=1\\ \therefore 原式成立当且仅当x\equiv y\,(mod\,p)\\ 又\because x,y\in [1,p-1]\\ \therefore x\equiv y\,(mod\,p)当且仅当x=y,与已知条件矛盾\\ \therefore 假设不成立，原命题成立假设存在xa≡ya(modp),且x=y∵gcd(a,p)=1∴原式成立当且仅当x≡y(modp)又∵x,y∈[1,p−1]∴x≡y(modp)当且仅当x=y,与已知条件矛盾∴假设不成立，原命题成立

证毕

# 第三个证明

接下来证明ap−1≡1(modp)a^{p-1}\equiv1\,(mod\,p)ap−1≡1(modp)

证：

∵1,2,⋯,p−1是modp的完全剩余系∴有1×2×⋯×p−1≡a×2a×⋯×(p−1)a(modp)即(p−1)!≡(p−1)!×ap−1(modp)又∵p是质数∴gcd((p−1)!,p)=1∴ap−1≡1(modp)\because 1,2,\cdots,p-1是\,mod\,p\,的完全剩余系\\ \therefore有1\times2\times\cdots\times p-1\equiv a\times2a\times\cdots\times (p-1)a\,(mod\,p)\\ 即 (p-1)!\equiv (p-1)!\times a^{p-1}\,(mod\,p)\\ 又\because p是质数\\ \therefore gcd((p-1)!,p)=1\\ \therefore a^{p-1}\equiv1\,(mod\,p)∵1,2,⋯,p−1是modp的完全剩余系∴有1×2×⋯×p−1≡a×2a×⋯×(p−1)a(modp)即(p−1)!≡(p−1)!×ap−1(modp)又∵p是质数∴gcd((p−1)!,p)=1∴ap−1≡1(modp)

证毕

# 回归正题

根据费马小定理，显然b'=b^

什么？不显然吗？

∵b′×b≡1(modP)bP−1≡1(modP)∴b′×b≡bP−1(modP)∴b′=bP−2\because b'\times b\equiv1\,(mod\,P)\\ b^{P-1}\equiv1\,(mod\,P)\\ \therefore b'\times b\equiv b^{P-1}\,(mod\,P)\\ \therefore b'=b^{P-2}∵b′×b≡1(modP)bP−1≡1(modP)∴b′×b≡bP−1(modP)∴b′=bP−2

那么\frac{a}{b}=a\times b^

#include<cstdio>

const int MOD = 19260817;

inline long long read() {
	long long X = 0, flag = 1; char ch = getchar();
	while(ch < '0' || ch > '9') {if(ch == '-') flag = 0; ch=getchar();}
	while(ch >= '0' && ch <= '9') {X = (X * 10 + (ch ^ 48)) % MOD; ch = getchar();}
	return flag ? X : -X;
}

long long a, b;

long long QuickPow(long long x, long long y) {
	long long ans = 1, k = MOD - 2;
	while(k) {
		if(k & 1) ans = ans * x % MOD;
		x = x * x % MOD;
		k >>= 1;
	}
	return ans * y % MOD;
}

int main() {
	a = read(); b = read();
	if(b) printf("%lld\n", QuickPow(b, a));
	else printf("Angry!\n");
	return 0;
}
